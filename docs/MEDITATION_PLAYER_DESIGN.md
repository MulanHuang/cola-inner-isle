# 🎨 冥想播放器 UI 设计方案

## 一、设计理念

打造一个**温柔、治愈、安心、灵性提升**的高端冥想播放器界面，参考 Calm、Headspace、Insight Timer 等顶级冥想应用的设计语言。

---

## 二、颜色方案

### 主色调 - 日落渐变系
```
背景渐变：
- 顶部：#FFF5F0 (奶油白)
- 中部：#FFE8E0 (柔和杏色)
- 底部：#FFD4C8 (浅珊瑚粉)

强调色：
- 主按钮渐变：#FFB6C1 → #FFA07A → #FF9999 (粉橙渐变)
- 进度条激活色：#FFB6C1 (浅粉)
- 进度条背景：rgba(255, 182, 193, 0.2) (半透明粉)

文字色：
- 标题：#4A3428 (深棕)
- 描述：#8B7355 (中棕)
- 时间/辅助文字：#A08D7A (浅棕)
- 按钮图标：#8B7355 (中棕)
```

### 设计原则
- 使用柔和的粉色、杏色、奶油色营造温暖治愈感
- 避免高饱和度颜色，保持视觉舒适
- 渐变过渡自然，无突兀感

## 四、视觉细节

### 1. 封面设计
- **尺寸**：480rpx × 480rpx
- **形状**：圆形 (border-radius: 50%)
- **阴影**：多层阴影营造立体感
  - 外阴影：`0 24rpx 80rpx rgba(255, 140, 140, 0.25)`
  - 中阴影：`0 8rpx 24rpx rgba(255, 182, 193, 0.2)`
  - 边框光晕：`0 0 0 1rpx rgba(255, 255, 255, 0.8)`
- **光晕动画**：3秒呼吸动画 (pulse)
- **旋转动画**：播放时 30秒匀速旋转

### 2. 进度条
- **高度**：4rpx (超细)
- **滑块**：16rpx 圆形
- **背景色**：rgba(255, 182, 193, 0.2) 半透明粉
- **激活色**：#FFB6C1 柔和粉
- **时间字体**：22rpx, font-weight: 300

### 3. 控制按钮
**主播放按钮**：
- 尺寸：128rpx × 128rpx
- 背景：三色渐变 `#FFB6C1 → #FFA07A → #FF9999`
- 阴影：多层立体阴影 + 内阴影
- 图标：52rpx 白色，带投影

**副按钮（快进/快退）**：
- 尺寸：96rpx × 96rpx
- 背景：白色到奶油色渐变
- 图标：36rpx 棕色
- 点击效果：scale(0.95)

### 4. 功能按钮（循环/倍速）
- 图标大小：44rpx
- 未激活：opacity: 0.4, grayscale(0.5)
- 激活状态：opacity: 1, scale(1.1)
- 文字：22rpx, font-weight: 300

## 六、动画效果

### 1. 封面光晕动画
```css
@keyframes pulse {
  0%, 100% { opacity: 0.6; transform: scale(1); }
  50% { opacity: 1; transform: scale(1.05); }
}
duration: 3s
timing-function: ease-in-out
iteration: infinite
```

### 2. 封面旋转动画
```css
@keyframes rotate {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
duration: 30s
timing-function: linear
iteration: infinite
```

### 3. 按钮交互
- 点击缩放：`transform: scale(0.95)`
- 过渡时间：`0.3s cubic-bezier(0.4, 0, 0.2, 1)`

## 七、设计亮点总结

### ✨ 为什么更高级？
1. **多层次渐变**：背景、按钮、阴影都使用精心调配的渐变
2. **立体感设计**：通过多层阴影和内阴影营造真实的立体效果
3. **微交互动画**：光晕呼吸、旋转、点击反馈，细节丰富
4. **统一的视觉语言**：间距、圆角、颜色系统高度统一

### 💆 为什么更疗愈？
1. **柔和色调**：粉色、杏色、奶油色营造温暖安全感
2. **低对比度**：避免刺眼，长时间观看不疲劳
3. **呼吸感动画**：光晕脉动模拟呼吸节奏，引导放松
4. **圆润设计**：所有元素圆角化，消除视觉攻击性

### 🎯 为什么更适合冥想？
1. **极简布局**：信息层级清晰，不分散注意力
2. **柔和光效**：模拟日落/烛光氛围
3. **细腻进度条**：不抢眼但清晰可见
4. **安静的配色**：不刺激，有助于进入冥想状态
