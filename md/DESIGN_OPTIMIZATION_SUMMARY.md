## 🎨 设计原则

### 视觉层次
1. **渐变装饰**: 为主要卡片添加径向渐变背景，增强视觉层次
2. **阴影系统**: 使用双层阴影，营造更真实的立体感
3. **圆角统一**: 统一使用 `20-24rpx` 圆角，保持设计一致性

### 交互反馈
1. **缩放效果**: 所有可点击元素添加 `:active` 缩放效果
2. **涟漪动画**: 快捷入口和按钮添加涟漪扩散效果
3. **过渡动画**: 使用 `cubic-bezier(0.4, 0, 0.2, 1)` 缓动函数

### 细节打磨
1. **毛玻璃效果**: 为状态标签和标签添加 `backdrop-filter`
2. **渐变线装饰**: 为卡片和导航栏添加细微的渐变线
3. **字间距优化**: 调整标题和标签的字间距，提升可读性

## 🔧 技术细节

### CSS 技巧
1. **双层阴影**: `box-shadow: 0 6rpx 20rpx ..., 0 2rpx 8rpx ...`
2. **毛玻璃**: `backdrop-filter: blur(20rpx)`
3. **涟漪效果**: 使用 `::after` 伪元素 + `transition`
4. **渐变装饰**: 使用 `::before` 伪元素 + `radial-gradient`

### 动画优化
1. 使用 `transform` 代替 `top/left`
2. 使用 `opacity` 代替 `visibility`
3. 使用 `cubic-bezier` 自定义缓动函数
