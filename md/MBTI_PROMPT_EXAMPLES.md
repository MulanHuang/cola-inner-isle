# 🎨 MBTI AI 解读 Prompt 示例

## 示例 1：INFJ 类型

### 输入数据

```json
{
  "type": "INFJ",
  "scores": {
    "E": 5,
    "I": 13,
    "S": 8,
    "N": 10,
    "T": 7,
    "F": 10,
    "J": 11,
    "P": 6
  }
}
```

### 维度分析

- 能量来源：E 5 : 13 I（强烈倾向于 I，占比 72%）
- 信息获取：S 8 : 10 N（略有倾向于 N，占比 56%）
- 决策方式：T 7 : 10 F（略有倾向于 F，占比 59%）
- 生活态度：J 11 : 6 P（明显倾向于 J，占比 65%）

### AI 生成示例

**核心特质总结**

你是一个 INFJ 类型的人，内心世界丰富而细腻。你既有独特的洞察力，能够看到事物背后的意义，也保持着对世界的温柔关注。你倾向于从内在获取能量，在独处中思考和感受，同时也能深刻地理解他人的情绪。

**能量与情绪模式**

你的能量更多来自内在世界。独处时，你能够充分恢复精力，思考和感受会变得更加清晰。你对情绪的感知很敏锐，既能理解自己的感受，也能共情他人的情绪。你可能会在社交后感到疲惫，需要一些安静的时间来重新充电。这不是逃避，而是你照顾自己的方式。

**人际与关系风格**

在人际关系中，你倾向于建立深度而真实的连接。你不追求广泛的社交，而是珍惜那些能够真正理解你的人。你善于倾听，也愿意给予支持，但有时可能会过度关注他人的需求而忽略自己。你的直觉帮助你理解他人的未说之言，这让你成为一个值得信赖的朋友。

**工作与学习风格**

你喜欢有意义、有深度的工作内容。你会为自己认同的目标全力以赴，但也需要足够的自主空间。你善于从整体视角思考问题，同时也能关注细节。你倾向于有计划地推进工作，但也保持着一定的灵活性。你可能会在追求完美和接受现实之间寻找平衡。

**温柔的成长建议**

1. 允许自己有独处的时间，这是你恢复能量的重要方式，不必为此感到内疚
2. 在表达自己时，可以更加直接一些，你的想法和感受值得被听见
3. 学会在理想与现实之间找到平衡，不必过于苛责自己或他人
4. 记得照顾好自己的身体和情绪，你的感受同样重要，不要总是把别人放在第一位
5. 相信自己的直觉，它往往能带你找到正确的方向，但也要给自己试错的空间

**温柔的成长建议**

1. 在做决定前，可以给自己一点时间思考长远影响，不必总是追求即时的结果
2. 尝试理解那些需要更多情感支持的人，有时候倾听比解决问题更重要
3. 适当地做一些计划，可以帮助你更好地实现长期目标
4. 记得给自己一些安静的时间，偶尔的独处也能带来新的洞察
5. 你的行动力是优势，但也要学会在必要时放慢脚步，照顾好自己的身心

4. 记得照顾好自己的情绪，不要总是把别人的感受放在第一位
5. 你的平衡感是一种天赋，允许自己在不同状态之间自然切换，不必强求一致

## Prompt 设计要点

### 1. 语言风格

✅ **推荐**：
- "你可能会..."
- "你倾向于..."
- "或许你会发现..."
- "这不是逃避，而是..."

❌ **避免**：
- "你应该..."
- "你必须..."
- "你的问题是..."
- "你需要改正..."

### 2. 结构要求

- 每段开头直接切入主题
- 使用具体的描述而非抽象概念
- 结合得分差异给出个性化分析
- 成长建议要温柔且可操作

### 3. 个性化要点

- **强烈倾向**（差值 >10）：重点描述这个特质
- **明显倾向**（差值 5-10）：作为主要特质描述
- **略有倾向**（差值 2-5）：提及平衡性
- **非常平衡**（差值 ≤2）：强调灵活性和多面性

## 🎯 使用建议

1. 根据实际效果调整 temperature（0.5-0.9）
2. 如果内容过于通用，增加更多个性化细节
3. 如果内容过于冗长,适当减少 max_tokens
4. 定期收集用户反馈，优化 Prompt
