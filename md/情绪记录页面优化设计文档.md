# 🌸 情绪记录页面优化设计文档

## 📋 优化概览

本次优化将"情绪记录"页面全面改造为一个温柔、疗愈、有陪伴感的情绪安全屋，强化感恩日记式引导，提升用户记录情绪和感恩内容的动机。

---

## 🎯 核心改进点

### 1. 页面结构优化

**新的记录流程：**
```
温暖一句（顶部卡片）
    ↓
情绪选择（12种情绪）
    ↓
情绪温柔反馈（自动生成）
    ↓
今日小确幸模块（感恩日记引导）
    ├─ 快速模板按钮（3个）
    └─ 文本输入区
    ↓
标签选择（12个标签，含感恩、小确幸）
    ↓
保存按钮 + 回顾记录按钮
    ↓
温柔回应按钮（独立区域）
    ↓
心岛的轻声回应（AI回复展示）
```

## 🎨 UI/视觉设计

### 配色方案

**主色调：温柔粉 + 奶油米 + 淡紫**
- 温暖一句卡片：`#fff5f7` → `#ffe8ec`（淡粉渐变）
- 情绪反馈卡片：`#fffbf0` → `#fff5e6`（奶油黄渐变）
- 感恩日记区：`#fffef9` → `#fffcf0`（米白渐变）
- 模板按钮：`#fff5f7` → `#ffe8ec`（淡粉渐变）
- 温柔回应按钮：`#e8d5ff` → `#d4bbff`（淡紫渐变）
- AI回应区：`#f5f3ff` → `#ede9fe`（浅紫渐变）
- 选中情绪：`#fff5f7` → `#ffe8ec`，边框 `#ffb6c1`
- 选中标签：`#ffb6c1` → `#ff8fa3`（粉色渐变）

### 圆角与阴影
- **卡片圆角：** 24rpx（更圆润）
- **按钮圆角：** 50rpx（完全圆角）
- **情绪/标签圆角：** 20rpx
- **阴影：** 更轻柔，透明度降低至 0.06-0.15

### 动画效果
1. **温暖一句图标：** 心跳动画（pulse，2秒循环）
2. **情绪反馈：** 淡入动画（fadeIn，0.5秒）
3. **按钮点击：** scale(0.95) 缩放反馈
4. **选中状态：** scale(1.02) 微放大

### 字体与间距
- **标题字号：** 32rpx，加粗 600
- **副标题字号：** 24rpx，颜色 #999999
- **正文字号：** 28rpx，行高 1.8-1.9
- **卡片内边距：** 36rpx 32rpx
- **卡片间距：** 24rpx

## 💻 技术实现要点

### 新增数据字段（emotion.js）
```javascript
emotionFeedback: "",  // 当前情绪的反馈文案
emotionFeedbacks: {   // 12种情绪的反馈映射
  happy: "真好，看到你开心的样子 ✨",
  // ... 其他11条
},
templates: [          // 3个快速模板
  { id: "gratitude", icon: "🙏", label: "今天感恩的事", text: "..." },
  // ... 其他2个
],
```

### 新增方法
1. **selectEmotion()** - 选择情绪时自动设置反馈文案
2. **insertTemplate()** - 点击模板按钮插入文本

### AI Prompt优化
```javascript
"你是一位温柔、善解人意的情绪陪伴者，名字叫'心岛'。
请用中文和我说话。请先共情、理解，再给一点温柔的建议或陪伴的话语。
不要指责，也不要简单地说'加油'。语气要像朋友一样温暖、真诚。
回应控制在100字以内，简短温暖即可。"
```
