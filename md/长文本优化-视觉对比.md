# 🎨 长文本优化 - 视觉对比

## 📊 优化前后对比

### 优化前（使用 input）

```
┌─────────────────────────────────────────┐
│ ① 今天我感激的是：家人的关心，同事的... │
│                                         │
└─────────────────────────────────────────┘

问题：
❌ 单行显示，长文本被截断
❌ 无法看到完整内容
❌ 无字数统计
❌ 无用户引导
```

### 优化后（使用 textarea）

```
┌─────────────────────────────────────────┐
│ ① 今天我感激的是：家人的关心，同事的帮  │
│   助，还有下班路上看到的美丽夕阳，让我  │
│   感到温暖和幸福。                      │
│                                         │
│   💬 写1-2句话就好        45/100       │
└─────────────────────────────────────────┘

优势：
✅ 多行显示，完整可见
✅ 高度自适应
✅ 实时字数统计
✅ 温柔用户引导
```

## 💝 设计细节

### 1. 对齐方式
- 输入项容器：`align-items: flex-start`（顶部对齐）
- 编号圆圈：`margin-top: 4rpx`（微调对齐）
- 字数统计：`margin-top: 8rpx`（与输入框间距）

### 2. 字数统计布局
- 使用 `flex` 布局
- `justify-content: space-between`（两端对齐）
- 左侧：提示文字（"💬 写1-2句话就好"）
- 右侧：字数统计（"45/100"）

### 3. 高度自适应
- 使用 `auto-height="{{true}}"`
- 最小高度：`44rpx`（约1行）
- 最大高度：`200rpx`（约5-6行）
- 超出最大高度后可滚动

### 4. 温柔的警告
- 不使用红色（#ff0000），使用温柔的粉色（#ff9a8b）
- 不使用"警告"、"超出"等强硬词汇
- 使用"还可以写 X 字"等温柔提示
