# 📦 情绪记录文案导入说明

## 🎯 目的

为情绪记录页面的"温暖一句"功能提供专属文案数据。

---

## 📄 数据文件

**文件路径：** `database/init-data/emotion-quotes.json`

**数据结构：**
```json
{
  "content": "文案内容",
  "author": "作者（可选）",
  "category": "emotion"
}
```

**文案数量：** 15 条

---

## 🚀 导入方法

### 方法 1：通过微信开发者工具导入（推荐）

1. 打开微信开发者工具
2. 进入「云开发控制台」
3. 选择「数据库」
4. 找到或创建 `quotes` 集合
5. 点击「导入」按钮
6. 选择 `database/init-data/emotion-quotes.json` 文件
7. 确认导入

### 方法 2：通过云开发控制台导入

1. 登录 [微信云开发控制台](https://console.cloud.tencent.com/tcb)
2. 选择你的环境
3. 进入「数据库」→「集合管理」
4. 找到 `quotes` 集合
5. 点击「导入」
6. 上传 `emotion-quotes.json` 文件
7. 确认导入

### 方法 3：手动添加（适合少量数据）

在云开发控制台的 `quotes` 集合中，点击「添加记录」，手动输入每条文案的内容。

---

## ✅ 验证导入

导入完成后，在云开发控制台执行以下查询验证：

```javascript
db.collection('quotes')
  .where({
    category: 'emotion'
  })
  .count()
```

应该返回 `15` 条记录。

---

## 🔍 查询示例

在小程序中查询情绪文案：

```javascript
const res = await db.collection('quotes')
  .where({
    category: 'emotion'
  })
  .aggregate()
  .sample({ size: 1 })
  .end();
```

## 🎨 扩展建议

如果你想添加更多文案，可以：

1. 编辑 `emotion-quotes.json` 文件
2. 添加新的文案对象（保持相同的数据结构）
3. 重新导入到云数据库

**注意：** 确保所有文案都包含 `"category": "emotion"` 字段，以便正确过滤。
